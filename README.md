# Leftover_Food_Redistribution_System1
Project Description
This system helps reduce food waste by connecting donors with people in need.

User Roles
Donor
Receiver
Admin
Features
User Registration & Login
Donor can add food
Receiver can request food
Admin can manage requests
Technologies Used
PHP
MySQL
HTML, CSS
How to Run the Project
Install XAMPP
Start Apache & MySQL
Open phpMyAdmin
Create a database named: leftover_food
Import the SQL file from /database folder
Place project folder inside htdocs
Run: http://localhost/Leftover_Food_Redistribution_System
🗂️ Database
File: leftover_food_db_2.sql
